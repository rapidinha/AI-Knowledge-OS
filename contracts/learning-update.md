# Contract: Learning Update

**Purpose:** Capture what was learned and what to update next.

## Required fields

| Field | Type | Notes |
|-------|------|-------|
| learned | string | New understanding |
| updates_paths[] | string[] | Wiki slugs or contract paths to revise |
| next_actions[] | string[] | Follow-up tasks |

## Emitters

Learning Agent (end of cycle).

## Consumers

Knowledge Curator, next Research cycle.

## Wiki touchpoints

- Updates `learning-paths`, `principles`, and `decision-logs` as needed.
