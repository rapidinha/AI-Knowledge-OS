# Knowledge domain (personal)

Structure only. Place personal notes under the subfolders — never promote this tree to upstream as-is.

| Path | Purpose | Upstream? |
|------|---------|-----------|
| `private/` | Personal knowledge that must never be promoted | Never |
| `shared/` | Material published elsewhere (blog, talk) — still not auto-promoted | Only via full rewrite into `wiki/` |
| `imported/` | Clippings and external imports | Never |

Default under uncertainty: `private/`.
